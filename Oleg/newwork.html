Создать класс объектов, у которых есть методы plusX, minusX.

При этом Х задается при создании экземпляра класса (объекта). Например, x = new Preobrazovatel(2, 0) Такой объект будет прибавлять или отнимать ДВА.

Методы plusX и minusX принимают массив массивов и возвращают копию, в которой массивам либо добавили либо убрали Х элементов.

Например:

x.plusX([ [1, 2, 3], [1, 2, 3] ]) // вернет [ [1, 2, 3, 0, 0], [1, 2, 3, 0, 0] ]

Для упрощения задачи.

1) Попробуйте создать класс, который создает объект с заданными свойствами

2) Попробуйте написать отдельные функции plus5, minus5, которые прибавляют или убирают по 5 элементов массива. Они должны возвращать новый массив
